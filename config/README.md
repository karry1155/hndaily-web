# Controlled configuration

- `hainan-administrative-divisions.json` controls article-level Hainan location IDs.
- `hainan-administrative-divisions.geojson` supplies the map outline.
- `topics.json` controls the single `topics.primary.category_id` value and gives
  the agent concise category boundaries.
- `page-sections.json` maps physical newspaper page names into stable archive sections.

Subjects, secondary topics, events and plans are discovered from each article;
they do not have finite candidate files.
