# AIHOT 最新变化：2026-07-11 → 2026-07-16

## 结论

旧参考库已经落后于公开站点。2026-07-12 的 AIHOT 2.0 重做同时改变了视觉、桌面时间线、手机导航、详情页和报告页；2026-07-13 又改变了精选代表信源规则；公开版本接口在 2026-07-15 继续更新热点与 Agent/API 合同。

## 视觉差异

- 品牌 Logo、墨青/砖红/琥珀色角色和浅深主题统一。
- 桌面端从开放新闻流变为卡片式时间线；手机端仍为连续列表。
- 手机首页固定四项底栏，当前热点、最新精选、筛选和新闻条目形成明确层级。
- 详情页强调长文阅读，AI 摘要折叠，原文/分享/收藏集中在顶栏。
- 日报增加今日看点、篇数、阅读时长和日/周/月切换。

## 手机端源码补齐

AIHOT 没有独立的手机站仓库；手机端由同一套 Next.js DOM、响应式 CSS 和客户端布局脚本驱动。本次快照新增：

- 5 个当前 CSS 文件，包含 `960px`、`640px` 断点、底栏、安全区、移动搜索和移动时间线规则。
- `layout-931cb53d8ec154f7.js`，包含移动底栏、主题切换、已读状态和更新日志未读状态。
- 收藏、时间线、日报、搜索、主题、收藏页、Agent 与关于页的公开功能脚本。
- 11 张按功能命名的手机/桌面截图。
